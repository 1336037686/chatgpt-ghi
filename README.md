## 项目介绍

> 使用ChatGPT自动获取每日GitHub热门项目，然后进行简单化翻译推荐。

**工作任务**
- [x] 1、实现github热门趋势爬虫;
- [x] 2、实现定时器，每天6点、12点、18点 抓取GitHub上热门趋势项目数据，并存库;
- [x] 3、创建定时器，对趋势项目通过调用chatgpt api进行项目分析，描述相关项目，便于学习理解;
- [x] 4、使用页面展示数据;
- [ ] 5、自动生成文章推送;
- [ ] 6、对接chatgpt接口，实现chatgpt聊天功能.
